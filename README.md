# The back end layer for the applictaion

## How to run? 🏃

#### Run the app ⚡
```sh
cd server/src
dotnet watch run -y
# open localhost:4000/swagger in your browser
```

#### Run the tests 🏁
```sh
cd server/test
dotnet test
```
